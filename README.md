# Инструкция по управлению сайтом Naprinte

Этот сайт является **премиальным статическим лендингом**, созданным на чистом коде (HTML5, CSS3, JavaScript). Это гарантирует максимальную скорость загрузки и надежность.

## 📂 Структура проекта
* `index.html` — основной файл сайта. Здесь находится весь текст и структура.
* `style.css` — файл стилей. Отвечает за дизайн, цвета и шрифты.
* `script.js` — файл логики. Отвечает за анимации и работу галереи.
* `img/` — папка со всеми изображениями.

## 📝 Как вносить изменения

### Изменение текста и цен
1. Откройте файл `index.html` с помощью любого текстового редактора (рекомендуется **Visual Studio Code** или **Sublime Text**).
2. Используйте поиск (`Ctrl + F`), чтобы найти нужный фрагмент текста.
3. Отредактируйте текст внутри тегов (например, `<h3>Текст</h3>`).
4. Сохраните файл.

### Замена изображений
1. Перейдите в папку `img`.
2. Чтобы заменить старое фото на новое, просто назовите новый файл **точно так же**, как старый (например, `interior_1.png`), и замените его в папке.
3. Сайт автоматически подтянет новое изображение.

## 🚀 Обновление сайта в сети (Hosting)
Если сайт размещен на **Netlify**:
1. Зайдите в панель управления Netlify.
2. Перейдите в раздел Deploys.
3. Перетащите обновленную папку с проектом в специальное поле загрузки. Изменения вступят в силу мгновенно.
